# 我的第一个 AI 项目

这是一个最小可运行的前端网页项目，使用 HTML、CSS 和 JavaScript 构建。首页针对手机端做了适配，并提供一个按钮，点击后会显示欢迎语。

## 项目结构

```text
.
├── index.html   # 页面结构
├── style.css    # 页面样式和移动端适配
├── script.js    # 按钮交互逻辑
└── README.md    # 项目说明
```

## 使用方法

1. 克隆或下载本项目。
2. 在浏览器中直接打开 `index.html`。
3. 点击页面中的“点击显示欢迎语”按钮，查看欢迎语。

也可以使用任意静态服务器运行，例如：

```bash
python3 -m http.server 8000
```

然后在浏览器访问 `http://localhost:8000`。
