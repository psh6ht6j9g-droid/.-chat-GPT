# 我的第一个 AI 项目

这是一个由 ChatGPT 帮你创建的最小可运行网页项目。

## 项目文件

- `index.html`：网页结构
- `style.css`：页面样式，已适配手机端
- `script.js`：按钮点击交互

## 怎么使用

1. 打开 GitHub 仓库页面。
2. 进入 `index.html` 文件。
3. 如果要在线预览，可以后续开启 GitHub Pages。

## 当前功能

页面中有一个按钮，点击后会显示欢迎语：

> 欢迎来到你的第一个 AI 网页项目！

## 下一步可以做什么

- 开启 GitHub Pages，让网页变成一个可访问的网址。
- 继续增加个人介绍、图片、日记、AI 工具入口等内容。
- 让 Codex 或 ChatGPT 继续帮你扩展页面功能。
